---
title: 'How to Spawn AI Traffic '
---
Spawning AI cars on a BeamMP server works a little differently than normal freeroam. Here's how you can add AI traffic to your BeamMP server using the Traffic app menu.

### Spawning AI Traffic

> **Note** The maximum amount of AI cars you can spawn is determined by the value of Max Cars Per Player. This can be adjusted in the **FSH Game Panel** under Server Startup.
>
> If you have CobaltEssentials installed, it MUST be configured; otherwise, it will override this setting with a default value of 1 for normal players.

1. Connect to your BeamMP server.

2. Press ESC and click UI Apps → Add App.

3. Search for Traffic, then select the Traffic app.

4. Close the UI Apps menu.

5. Spawn cars using the new Traffic menu on the left-hand side.

There are multiple ways to configure these cars. You can set a speed limit, configure the number of cars that spawn, set their AI Mode to Chase, Flee, Random, Traffic, and more.

Once finished, you can close this menu by navigating to UI Apps once more, selecting Edit Apps, then clicking the X to close the Traffic menu.
