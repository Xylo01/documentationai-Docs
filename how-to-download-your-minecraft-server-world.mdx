---
title: download your Minecraft server world
---
This guide explains how to manually backup and download your Minecraft world using the file manager in the panel. Backing up only the world is useful if you want to transfer your map to another server, share it with others, or keep a copy of your progress without downloading the entire server files. If you need to preserve plugins, mods, and server configurations, you should instead create a full server backup.

---



1. Access your server from the <a href="https://gamepanel.fourseasonshosting.com/" target="_blank" rel="noopener noreferrer">gamepanel</a>

1) Stop your server to prevent missing any progress.

1. From the left side of the panel, open the **Files** tab.

1) Select your world directory and press the **Archive** button.
