---
title: Uploading Files Via SFTP
---
**SFTP (Secure File Transfer Protocol)** is a network protocol used for transferring files between a client and a server. It serves as an alternative to browser-based file managers, offering a safe and more reliable method for accessing, uploading, and downloading large files from your server.

---

We recommend using **FileZilla**, however, you may use a different application.

1. Download and install the appropriate version of <a href="https://filezilla-project.org/download.php?show_all=1" target="_blank" rel="noopener noreferrer">FileZilla client</a> for your computer.

2. Login to the <a href="https://gamepanel.fourseasonshosting.com" target="_blank" rel="noopener noreferrer">game panel</a> and select your server.

3. From the left side of the game panel, navigate to the **Settings** page and copy your SFTP credentials.

1. Paste your credentials to FileZilla and press “Quickconnect”.

<Image src="https://blob-cdn.documentation.ai/org-126413f3-60b3-418a-a6a3-c91731a9859b/doc-5ddb8241-3292-4172-aaf3-e39fd0e3f428/1777424715571-ypxa5q0qocp-pasted-image-1777424713710.png?fm=auto&auto=compress%2Cformat" width="769" height="144" alt="" />

<Callout kind="info" collapsed="false">
  *Note:*

  - *Your SFTP password is the same as your game panel password.*

  - *The port is specified by the numbers at the end of the server address.*
</Callout>
