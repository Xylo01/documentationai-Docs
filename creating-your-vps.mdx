title: Creating Your VPS
---
Here we will set up your new VPS together! On the new panel and go through some of the basics of the panel while setting up

<Image src="https://blob-cdn.documentation.ai/org-126413f3-60b3-418a-a6a3-c91731a9859b/doc-5ddb8241-3292-4172-aaf3-e39fd0e3f428/1777425648155-m95xfkk5gdo-pasted-image-1777425647235.png?fm=auto&auto=compress%2Cformat" width="1484" height="829" alt="" caption="When Logging In to the VPS Dash it will look something like this." />

1\) Now we can see our server is "Awaiting Setup..." so let's click that big "Manage" button to start the set up of the VPS

2\) In this GIF you can see that we named our server and set the hostname to "myserver.fsh"

<Image src="https://blob-cdn.documentation.ai/org-126413f3-60b3-418a-a6a3-c91731a9859b/doc-5ddb8241-3292-4172-aaf3-e39fd0e3f428/1777425673223-xdvsewdfn3p-pasted-image-1777425672409.png?fm=auto&auto=compress%2Cformat" width="1660" height="675" alt="" caption="Name and Host Name of your server" />

3\) Now we can install our Operating System (OS).

<Callout kind="danger" collapsed="false">
  We recommend with any Linux install you use a SSH Key! it's not required but it's good practice for security of your server
</Callout>

4\) Click on the Group of the OS you would like to use.. For this demo we will use Windows Server 2019.

<Callout kind="info" collapsed="false">
  Clicking on Advanced Options allows you to change your DNS (auto set to Google DNS) and Enable/Disable VNC, this can also be done afterwords on the Dashboard
</Callout>

4\) Now let install the OS

<Image src="https://blob-cdn.documentation.ai/org-126413f3-60b3-418a-a6a3-c91731a9859b/doc-5ddb8241-3292-4172-aaf3-e39fd0e3f428/1777425690659-lvm8l6qufc-pasted-image-1777425689929.png?fm=auto&auto=compress%2Cformat" width="1569" height="656" alt="" caption="Installing Windows Server 2019" />

<Callout kind="alert" collapsed="false">
  Some OS like Windows could take some time installing you may see a popup warning you of this, you can click "Install Now" and continue the install
</Callout>

<Image src="https://blob-cdn.documentation.ai/org-126413f3-60b3-418a-a6a3-c91731a9859b/doc-5ddb8241-3292-4172-aaf3-e39fd0e3f428/1777425748170-3yt92k03zgy-pasted-image-1777425747213.png?fm=auto&auto=compress%2Cformat" width="1005" height="421" alt="" caption="Installing Your OS" />

<Callout kind="danger" collapsed="false">
  If deployer comes back with a failed task please open a ticket on our Discord or in the Client Portal
</Callout>

Once the installation is completed, you will be redirected to the Dash of that VPS server and the VPS login will be emailed to you (check junk folder)
