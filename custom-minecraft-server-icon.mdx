---
title: Custom Minecraft Server Icon
---
Adding a custom icon to your Minecraft server list entry is a great way to make your server stand out and attract new players. Here’s a simple guide to setting up your server icon.

### Server Icon Specifications

Before you start, make sure your icon meets these requirements:

- **Size**: The icon must be 64x64 pixels you can use this <a href="https://imageresizer.com/" target="_blank" rel="noopener noreferrer">image resizer</a> to resize your image if needed

- **Format**: The icon file should be saved in PNG format.

- **File Name**: Save your icon with the name `server-icon.png`.

### Use the built-in game panel tool to upload an icon

Use our built-in tool to upload your icon and set it up on your server, it is located in the "Files" tab

### Upload and Set the Icon on Your Server

Once your icon is ready and meets the specifications, follow these steps to upload and set it:

1. **Save the Icon**: Confirm your file is named `server-icon.png` and is 64x64 pixels in size.

2. **Upload the Icon**: In your server’s main directory, upload the `server-icon.png` file (navigate to Files -> Upload).

3. **Restart the Server**: Restart your server to apply the changes and display your new icon.
