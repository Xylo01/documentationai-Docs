---
title: upload a custom Minecraft world
---
Before proceeding, kindly ensure that the world you want to upload is compatible with the Minecraft version of your server. We also recommend taking a backup of your current world if you want to save the progress for future use.

---

1. Upload the ZIP file to your server using <a href="https://docs.fourseasonshosting.com/docs/panel-knowledge-base/game-server-panel/uploading-files-via-sftp" target="_blank" rel="noopener noreferrer">SFTP</a>

2. Access your server from the <a href="https://gamepanel.fourseasonshosting.com/" target="_blank" rel="noopener noreferrer">gamepanel</a> and select the **files** tab.

3. Unarchive the ZIP file using the panel's function. Make sure that the world folder is located in the root directory, with the content directly inside it.

<Image src="https://blob-cdn.documentation.ai/org-126413f3-60b3-418a-a6a3-c91731a9859b/doc-5ddb8241-3292-4172-aaf3-e39fd0e3f428/1777425898872-wn04o08x10c-pasted-image-1777425898003.png?fm=auto&auto=compress%2Cformat" width="1198" height="410" alt="" />

Now we need to specify the name of the world folder so the server knows which one to use.

1. Select the **Config Editor** tab and open the **Server Properties** section.

2. Modify the `level-name` property to match the name of the world folder that you previously uploaded. For this example, we have used "house-map", yours will vary.

<Image src="https://blob-cdn.documentation.ai/org-126413f3-60b3-418a-a6a3-c91731a9859b/doc-5ddb8241-3292-4172-aaf3-e39fd0e3f428/1777425908535-ysbh2lsfm1k-pasted-image-1777425907542.png?fm=auto&auto=compress%2Cformat" width="2304" height="1049" alt="" />

1. Press **save** at the end of the page, then restart your server.
