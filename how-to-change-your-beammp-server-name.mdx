---
title: How to Change BeamMP Server Name
---
This is what you do if you need to change your server name on your BeamMP server, also referred to as a hostname. This is what is displayed to players in the server browser.

### Changing Your BeamMP Server Name

1. Log in to the **FSH Game Panel**.

2. Select your server.

3. In the sidebar, go to **Startup**.

4. Find the Server Name setting and enter your desired server name.

5. Restart your server.

> **Note** This setting will override anything you have manually set inside your config file.
